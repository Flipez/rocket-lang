# Local Variables
Variables represent the object as a memorable name.

```js
a = "test"
```
## Examples

```js
some_integer = 1;
name = "RocketLang";
array = [1, 2, 3, 4, 5];
some_boolean = true;
```

Also expressions can be used
```js
another_int = (10 / 2) * 5 + 30;
an_array = [1 + 1, 2 * 2, 3];
```
